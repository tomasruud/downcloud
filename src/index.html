Dear Downcloud user

Due to recent changes in Soundcloud APIs, this app is left in a rather broken state. This leaves me with two possible paths regarding the future for this application:

  1. Cease any further development and let people use the download track functionality that is already built into the current version of Soundcloud.
  2. Migrate this application out of the browser, to a command line client for instance, and continue development.

Personally I'm not using this tool myself anymore, but for anyone interested in continuing to use a tool like this, I suggest dropping a comment or reaction over at the ticket I've set up on Github . If there is enough interest, I would like to continue doing some work on this, but it depends on the demand.

Cheers,
